Explique o que é o Grid Layout

O Grid Layout é um modelo de layout em CSS que permite criar layouts de grade complexos e alinhados em páginas da web. Ele oferece um sistema de grade bidimensional, permitindo que você organize elementos em linhas e colunas, criando designs altamente flexíveis e responsivos.

Cite e dê exemplos das propriedades do Grid Container

  • display : grid

    Define um elemento HTML como um contêiner de grade.

    Exemplo

    .grid-container { display: grid }
  • grid-template-columns e grid-template-rows

    Define o número e o tamanho das colunas e linhas da grade.

    Exemplo

    .grid-container { display: grid; grid-template-columns: 1fr 2fr 1fr; grid-template-rows: auto 100px; }
  • grid-template-areas

    Define áreas nomeadas dentro da grade para facilitar o posicionamento dos elementos filhos.

    Exemplo

    .grid-container { display: grid; grid-template-areas: "header header" "sidebar main" "footer footer"; }
  • grid-gap

    Define o espaçamento entre as colunas e linhas da grade.

    Exemplo

    .grid-container { display: grid; grid-gap: 20px; /* Espaçamento de 20 pixels entre as colunas e linhas */ }
  • justify-items e align-items

    Define o alinhamento padrão dos elementos dentro das células da grade.

    Exemplo

    .grid-container { display: grid; justify-items: center; /* Alinhamento horizontal central */ align-items: center; /* Alinhamento vertical central */ }
  • grid-auto-columns e grid-auto-rows

    Define o tamanho automático das colunas e linhas quando elementos são adicionados dinamicamente à grade.

    Exemplo

    .grid-container { display: grid; grid-auto-columns: 100px; /* Colunas com 100px de largura automática */ grid-auto-rows: 50px; /* Linhas com 50px de altura automática */ }

Cite e dê exemplos das propriedades do Grid Item

  • grid-colum e grid-row

    Define a posição do item na grade em relação às colunas e linhas.

    Exemplo

    .grid-item { grid-column: 2 / 4; /* Ocupa da coluna 2 à coluna 4 */ grid-row: 1 / 3; /* Ocupa da linha 1 à linha 3 */ }
  • grid-area

    Especifica em qual área nomeada da grade o item deve ser colocado.

    Exemplo

    .grid-item { grid-area: main-content; /* Coloca o item na área nomeada "main-content" */ }
  • justify-self e align-self

    Define o alinhamento horizontal e vertical do item dentro de sua célula na grade.

    Exemplo

    .grid-item { justify-self: center; /* Alinhamento horizontal central */ align-self: end; /* Alinhamento vertical no final */ }
  • order

    Controla a ordem de exibição dos itens na grade. Itens com valores menores aparecem primeiro.

    Exemplo

    .grid-item { order: 2; /* Este item aparecerá depois do item com order: 1 */ }
  • grid-column-start, grid-column-end, grid-row-start, grid-row-end

    Define as posições iniciais e finais das colunas e linhas em que um item deve ser colocado.

    Exemplo

    .grid-item { grid-column-start: 2; /* Começa na coluna 2 */ grid-column-end: 4; /* Termina na coluna 4 */
    grid-row-start: 1; /* Começa na linha 1 */ grid-row-end: 3; /* Termina na linha 3 */ }

O que é a unidade fr

A unidade "fr" em CSS (unidade de fração) é uma unidade de medida relativa usada em layouts de grade (Grid Layout)
e em layouts flexíveis (Flexbox). Ela representa uma fração do espaço disponível dentro de um contêiner flexível ou de grade.


Exemplo

.grid-item { width: 3fr; }

O que é a notação repeat

A notação `repeat` é uma funcionalidade do CSS Grid Layout que permite definir padrões repetitivos para as colunas e/ou linhas de uma grade.
Ela é especialmente útil quando você deseja criar layouts de grade com várias colunas ou linhas que têm a mesma largura ou altura.


Exemplo

.grid-container { grid-template-columns: repeat(3, 1fr); }

O que são as linhas do Grid

As linhas do grid são os elementos que dividem o grid tanto horizontal quanto verticalmente,
criando células onde você pode colocar conteúdo


Exemplo

.grid-container { grid-template-rows: repeat(2, 1fr); }

O que é uma célula no Grid

Uma célula no grid é uma área retangular específica dentro da estrutura do grid.
Ela é formada pela interseção de uma linha de coluna e uma linha de linha no grid.
Essas células são usadas para organizar e posicionar elementos de conteúdo em um layout de grade.


Exemplo

.grid-item { grid-column: 2; grid-row: 1; }

O que são áreas do Grid

As "áreas" em CSS Grid Layout são regiões nomeadas dentro do contêiner de grade onde você pode posicionar elementos de forma específica.
Elas são uma maneira conveniente de definir e organizar o layout da grade, permitindo que você atribua nomes a áreas específicas da grade e, em seguida, coloque elementos nessas áreas pelo nome


Exemplo

.grid-container { grid-template-areas: "header header header" "sidebar main main" "footer footer footer"; }
.header { grid-area: header; } .sidebar { grid-area: sidebar; } .main { grid-area: main; } .footer { grid-area: footer; }
